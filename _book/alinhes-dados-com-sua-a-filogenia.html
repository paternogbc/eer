rownames(dados) <- dados$Species

Primeiro utilize a função name.check para verificar se as espécies presentes no banco de dados estão na filogenia e se as espécies presentes na filogenia estão no banco de dados. Ok representa que os dados batem perfeitamente com a filogenia

name.check(arvore, dados) 
## [1] "OK"

Caso alguma espécie da tabela de dados não esteja na filogenia ou alguma espécie da filogenia não estaje no bando de dados, esta função irá mostrar quais são essas espécies. Veja o exemplo abaixo (removendo uma espécie da filogenia e uma espécie do bando de dados)

# Sem a espécie "Catoptrophorus_semipalmatus" na filogenia:
name.check(drop.tip(arvore, "Catoptrophorus_semipalmatus"), dados)
## $tree_not_data
## character(0)
## 
## $data_not_tree
## [1] "Catoptrophorus_semipalmatus"
# Sem a espécie "Catoptrophorus_semipalmatus" nos dados:
name.check(arvore, dados[dados$Species != "Catoptrophorus_semipalmatus", ])
## $tree_not_data
## [1] "Catoptrophorus_semipalmatus"
## 
## $data_not_tree
## character(0)

Caso o seu banco de dados ou filogenia tenham espécies não congruentes, você precisa cortar essas espécies do banco de dados e/ou da filogenia: Os passos a seguir garantem que os dados e a filogenia estejam congruentes. Apenas utilize esses passos caso name.check() não der “OK”.

Para ilustrar, vamos criar o banco de dados e uma árvore com alumas espécies faltando:

arvore2    <- drop.tip(arvore, 1:4) # Remove espécies de 1:4 da árvore
dados2     <- dados[-(1:7), ]       # Remove espécies de 1:7 do bando de dados

name.check() para detectar as diferenças:

nao.cruza  <- name.check(arvore2, dados2)
nao.cruza # Veja o que está faltando na filogenia e no banco de dados:
## $tree_not_data
## [1] "Actophilornis_africanus" "Aphriza_virgata"        
## [3] "Arenaria_interpres"      "Arenaria_melanocephala" 
## [5] "Bartramia_longicauda"    "Calidris_alba"          
## [7] "Calidris_alpina"        
## 
## $data_not_tree
## [1] "Catoptrophorus_semipalmatus" "Tringa_flavipes"            
## [3] "Tringa_ochropus"             "Tringa_stagnatilis"

Espécies presentes em ambos (dados/filogenia):

ambos <- intersect(dados2$Species, arvore2$tip.label)
ambos

União das espécies que não cruzam simultaneamente na árvore e nos dados:

remover <- c(nao.cruza$tree_not_data, nao.cruza$data_not_tree)
remover
##  [1] "Actophilornis_africanus"     "Aphriza_virgata"            
##  [3] "Arenaria_interpres"          "Arenaria_melanocephala"     
##  [5] "Bartramia_longicauda"        "Calidris_alba"              
##  [7] "Calidris_alpina"             "Catoptrophorus_semipalmatus"
##  [9] "Tringa_flavipes"             "Tringa_ochropus"            
## [11] "Tringa_stagnatilis"
# Quantas espécies serão removidas?
length(remover)
## [1] 11
# Quantas espécies serão mantidas na filogenia e nos dados:
length(ambos)
## [1] 60

Agora sabemos quais espécies terão que ser excluídas da filogenia e do banco de dados para deixar ambos perfeitamente alinhados. Vamos remover estas espécies.

# Corta espécies da árvore:
arvore2 <- drop.tip(arvore2, remover)
# Corta espécies dos dados:
dados2 <- dados2[!dados2$Species %in% remover, ] 
# Verifica se deu certo:
name.check(arvore2, dados2)
## [1] "OK"

Por fim, verifique se a ordem das espécies é a mesma no banco de dados e na filogenia:

sum(dados2$Species == arvore2$tip.label) == length(dados2$Species)
## [1] FALSE

Se der “FALSE” é pq a ordem das espécies não é extamente a mesma entre filogenia e os dados.

Para colocar filogenia e dados na mesma ordem:

# Ordenando a ordem das espécies no banco de dados
tip.order <- match(arvore2$tip.label, rownames(dados2))
dados2 <- dados2[tip.order, ]
# Agora repita o teste (tem que dar "TRUE")
sum(dados2$Species == arvore2$tip.label) == length(dados2$Species)
## [1] TRUE

4.1.2 Pacote caper

O pacote ‘caper’ possui a função comparative.data() que serve para juntar os dados e a filogenia em um único objeto. Essa função garante que apenas espécies presentes na filogenia e nos dados serão mantidas no objeto final.

## Utilizando a árvore original:
comp.dados <- comparative.data(arvore, dados, 'Species')
print(comp.dados)
## Comparative dataset of 71 taxa:

##    71 tips, 62 internal nodes
